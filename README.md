Built a dynamic newspaper web application using Next.js, SQLite, CSS, and JavaScript. Implemented features to browse latest news and filter by date, allowing users to select a specific year and month to view archived news. Leveraged Next.js API routes for data management and improved performance with SSR and SSG rendering techniques.
